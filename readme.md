專案建立
方式
1. 確認網站所需風格與樣式
2. 於ps中繪製手稿草稿
3. 於AI中製作向量ICON
4. 在bootstrap裡尋找適合的模板
5. 變更模板區塊背景色並且將區塊分配好與手稿一致
6. 更改內文圖片與文字
7. 確認超鏈接與按鈕鏈接至何處，並且修正
8. 於bootstrap中加上網頁動態效果
9. 反復測試於瀏覽器及手機內的效果.
10. 精緻化細節(例如網站名稱/縮圖等)
11. 進行問題修正
12. 測試完無誤，上架git hub
